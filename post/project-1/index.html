Project 1: Arknights Class Clustering

By Baihan Yang

A personal project exploring the character classes in a mobile game called Arknights after learning about unsupervised learning techniques.

  • Cleaned and engineered data using domain knowledge
  • Used a pipeline to scale data and implement KMeans algorithm to classify our game characters into groups
  • Created a T-SNE mapping to visualize our clusters and see our results
  • Experiment with different features to see if we can improve our result
  • Vary the amount of clusters to see if we can pick up on different feature groupings Link to Kaggle notebook
